# App Icons for PWA

This folder should contain the following PWA icon files:

## Required Icons

1. **icon-192x192.png** - 192x192 pixels
2. **icon-512x512.png** - 512x512 pixels

## How to Generate Icons

### Option 1: Using Online Tools
- Visit: https://www.pwabuilder.com/imageGenerator
- Upload your logo/icon (minimum 512x512 px)
- Download the generated icons
- Place them in this folder

### Option 2: Using Figma/Photoshop
- Create a 512x512 px canvas
- Design your app icon (use your brand colors)
- Export as PNG:
  - 192x192 px â†’ `icon-192x192.png`
  - 512x512 px â†’ `icon-512x512.png`

### Option 3: Quick Placeholder (For Testing)
- Use a solid color square with text "GST"
- Generate using: https://via.placeholder.com/512x512/000000/FFFFFF?text=GST
- Save as both sizes

## Icon Guidelines

- **Safe Zone**: Keep important content within 80% of the icon area
- **Maskable**: Icons should work with rounded corners/circles
- **Format**: PNG with transparency
- **Content**: Should represent your GST Filing app (calculator, document, GST logo)
